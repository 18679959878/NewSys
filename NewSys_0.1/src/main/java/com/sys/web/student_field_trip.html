江西科技师范大学

专业（生产）实习实习学生的有关规定

一、实习生的基本要求

  （一）凡无故不参加专业（生产）实习和专业（生产）实习不及格者，不发毕业证，只发结业证。

  （二）因特殊原因经系领导及教务处批准未参加专业（生产）实习的学生，必须补生产实习，其时间及方式由系研究后报教务处批准。

  （三）实习生应该具备的素养：

  1.了解党的教育方针、政策、忠于人民教育事业、热爱教育工作。

  2.修完选修基础课、主要专业课，掌握本专业的基础理论和基本技能。

  （四）实习生具有双重身份，既是江西科技师范大学的学生，又是实习单位的新职工，要虚心接受双方指导教师的指导。

  （五）实习生要严格遵守“实习生守则”以及实习单位规章制度。

  （六）对实习单位的意见和建议，要通过领队教师有组织地提出。

二、实习生守则

  （一）实习生应该按教学计划规定完成全部实习活动。实习期间要严肃认真，刻苦钻研，客服苦难，团结互助，努力完成各项实习工作，争取优良成绩。

  （二）自觉遵守实习工作和实习单位的一切规章制度，服从实习单位的统一领导。

  （三）未经公布的正式决定，不得向实习单位透露。向实习单位借用的各种材料应该认真保管，不得遗失和外传，并按时归还实习单位。

  （四）实习生按每日作息制度进行实习工作，未经指导教师批准，不得迟到早退或缺席。实习期间不能无故回家探亲访友，如因故需请假，须有确切证明。

  （五）实习生之间应发扬团结互助，批评和自我批评的精神，互相帮助，互相学习，共同提高。

  （六）实习生实习期间不准谈恋爱。如有违反，必须及时从严处理。

  （七）实习生在实习期间，不能邀请或接受邀请，与实习单位职工到校外进行与实习工作无关的活动。

三、实习生请假制度

  （一）实习生因病、因事不能参加实习活动，必须办理请假手续。

  （二）请假半天以内者，经本系带队指导教师同意；请假一至二天并外出者，需有请假报告，经带队指导教师签加意见，由实习单位工作组批准，方能离队。

  （三）因病或特殊原因，请假三至七天着，需报经系生产实习指导小组集体研究审批，并报单位教务处备查。请假八天以上者由系生产实习指导小组省察，签署意见后，报校实习指导委员会批准。凡不请假或请假未经批准者，作擅自离队处理。

  （四）无故缺勤一天，按旷课8课时处理。无故缺勤七天以上者，取消其实习成绩，需补实习：请事、病假十天以上者，亦需补实习。

四、实习生纪律

  （一）坚持四项基本原则，反对资产阶级自由化。

  （二）深入了解基层，工作认真负责，努力克服困难，全面完成各项实习任务。

  （三）严格遵守实习工作规定和实习单位的一切规章制度。

  （四）服从领导，虚心接受双方指导教师的指导，不准随便议论实习单位的教师和工作。

  （五）严格要求以身作则，教书育人，以“五讲四美”作学生的表率。

  （六）着装整齐、朴素、举止端庄大方。

  （七）作风要正派，语言要文明。

  （八）实习期间一般不准请假，离开实习单位必须经实习单位实习领导小组批准。

  （九）如有违反以上纪律者，视情节轻重予以严重处理。
